# 📊 Superstore Sales Analysis Dashboard (Power BI)

## 📌 Overview
This project is an interactive Power BI dashboard built to analyze the **Superstore Sales dataset**.  
It provides insights into sales performance, profit trends, shipping behavior, and customer segments to support data-driven business decisions.

---

## 🎯 Objective
The main goal of this project is to transform raw sales data into meaningful insights that help understand:
- Sales performance across different regions
- Profitability by category and sub-category
- Customer purchasing behavior
- Shipping efficiency and delivery trends

---

## 🛠️ Tools & Technologies
- Power BI
- DAX (Data Analysis Expressions)
- Data Cleaning & Transformation
- Data Visualization

---

## 📊 Key Features
- 📈 Sales & Profit analysis by region, category, and segment
- 🧾 Order trends over time
- 🚚 Shipping mode and delivery insights
- 💰 Profit margin analysis
- 🎛️ Interactive filters and slicers for dynamic exploration

---

## 🔍 Key Insights
- Technology category shows the highest profit margins
- Certain regions consistently outperform others in sales
- Shipping delays can impact customer satisfaction and profitability
- Seasonal trends affect overall sales performance

---

## 📁 Project Files
- `Superstore_Sales.pbix` → Main Power BI dashboard file
- `Superstore sales dataset.csv` → The Dataset
- `Superstore_Dashboard_Theme_Final.json` → Theme of the project


---

## 🚀 How to Use
1. Download the `.pbix` file
2. Open it using Power BI Desktop
3. Explore the dashboard using filters and visuals
